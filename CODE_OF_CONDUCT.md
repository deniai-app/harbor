# Code of Conduct

This project is committed to a welcoming and respectful community.

## Our Standards

- Be respectful and considerate in all communications.
- Keep feedback constructive.
- Avoid harassment, discrimination, or abusive language.
- Respect different skill levels and perspectives.

## Enforcement

Project maintainers may remove content, warn contributors, or restrict participation if behavior violates this conduct.

## Scope

This policy applies to all project spaces, including:

- Issues and pull requests
- Chat, comments, and code review contexts
- Any official project channels

Thank you for helping us keep this project healthy and open.
